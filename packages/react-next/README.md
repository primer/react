## Getting started

To install `@primer/react-next` in your project, you will need to run the following
command using [npm](https://www.npmjs.com/):

```bash
npm install -S @primer/react-next @primer/primitives @primer/tailwind-config
```

## What's Different?

The `@primer/react-next` package is an exploration of building the primer design system with a separate set of fundamentals, leveraging community created primitives for Components and an extensible styling system built on tailwindcss.
